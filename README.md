# CheesDex
## _The Database for all Chess.com Players_



ChessDex is a single page application that allows the user to:

- See the top three and top ten players on Chess.com along with their stats.
- Search for specific players by their username.
- Save the players.

## Features

- Search for a player and a card will be displayed with their stats.
- Click on the "Profile" button to go to the player's Chess.com profile.
- Click on the "Follow!" to add the player card to a carousel at the bottom of the page.
- Click on the "Delete" button to remove player from the carousel


## Tech

ChessDex uses the Twitter Bootstrap open source project.

The API used and documentation can be found at https://www.chess.com/news/view/published-data-api
